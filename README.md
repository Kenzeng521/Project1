# NYC Taxi Data Analysis with MPI
This project implements parallel processing for NYC taxi datasets using MPI.

## How to Run
1. Install MPI (e.g., OpenMPI).
2. Run the script: `mpirun -np 4 python mpi-youhua3.py`

## Note on Data
The original datasets (`.csv`) are too large for GitHub and are excluded. 
Please place the NYC Taxi 2022 dataset in the root folder before running.
